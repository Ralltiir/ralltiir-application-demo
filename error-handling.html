Error Handling

Upon network error, or inlined-script error, Ralltiir will navigate to the page directly (e.g. by `location.href = xxx`).

The url of the target page contains `rt-err` and `rt-msg` params which contains detailed information about the error. See following examples, note the result URL:

  • A Page Not Exist
  • Bad .rt-head
  • Bad .rt-body
